# Stocks API

A real-time stock price tracker built with Python and Flask.

## Features
- Live stock price data
- Daily change and % change
- Day high and low
- Color coded green/red for gains and losses
- Error handling for invalid ticker symbols

## Tech Stack
- Python
- Flask
- Alpha Vantage API
- HTML/CSS/JavaScript

## Live Demo
https://stocks-api-8v1b.onrender.com

## Setup
1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Get a free API key at alphavantage.co
4. Create a `.env` file and add `ALPHA_VANTAGE_KEY=your_key_here`
5. Run `python app.py`
